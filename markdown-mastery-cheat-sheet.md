# Markdown Mastery Cheat Sheet

## 1. Headings
```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## 2. Text Formatting
```md
**bold**
*italic*
***bold italic***
~~strikethrough~~
`inline code`
```

## 3. Paragraphs & Line Breaks
```md
First paragraph.

Second paragraph.
```

Tip: Add two spaces at end of line for a line break
without starting a new paragraph.

## 4. Lists
### Unordered:
```md
- Item 1
- Item 2
  - Sub-item
```
### Ordered:
```md
1. First
2. Second
```
### Task List (GFM):
```md
- [x] Done
- [ ] To-do
```
